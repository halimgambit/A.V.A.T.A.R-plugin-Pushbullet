# Pushbullet
Pushbullet connecte A.V.A.T.A.R à votre réseau mobile pour envoyer vos messages instantanément.

This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework.

🎯 Usage
Commandes :

envoie un message à [contact],
envoie un sms à [contact]

https://docs.pushbullet.com/

# Multi-room
The Pushbullet plugin is fully multi-room.

# Multi-language
The Pushbullet plugin relies solely on the system's available languages.
